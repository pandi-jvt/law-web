# Case Management Frontend

React 18 + TypeScript Create React App frontend using Material UI.

## Run

```bash
npm install
REACT_APP_API_BASE_URL=http://localhost:9000 PORT=3005 npm start
```

## Build

```bash
REACT_APP_API_BASE_URL=https://api.example.com npm run build
```

Output directory: `build/`.

## Important Files

- `src/contexts/AuthContext.tsx`
- `src/services/api.ts`
- `src/utils/permissions.ts`
- `src/routes/MainRoutes.tsx`
- `src/pages/Cases.tsx`

This app uses MUI and CRA.
